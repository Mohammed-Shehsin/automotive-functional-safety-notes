# Risk Reduction Techniques

Risk reduction can be achieved through product measures and process measures.

## Product Measures

- redundancy
- diagnostics
- self-tests
- fail-safe mechanisms
- defensive programming
- modular architecture

## Process Measures

- design methods
- analysis techniques
- test methods
- configuration management
- safety case
- documentation

## STM32 Practical Examples

- watchdog supervision
- brownout reset
- CRC verification
- ADC plausibility checks
- startup self-test
- communication timeout handling
