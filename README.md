# Automotive Functional Safety Notes

This repository documents my self-study of automotive functional safety, mainly focused on ISO 26262, embedded systems, and safety-oriented design practices.

The goal is not to present myself as a functional safety expert, but to show a structured engineering learning process connected to my background in embedded systems, automation, robotics, PCB debugging, and industrial troubleshooting.

## Purpose

- Document ISO 26262 learning progress
- Build beginner-to-intermediate functional safety knowledge
- Connect safety concepts with embedded systems and ECU design
- Prepare for embedded/automotive engineering interviews
- Create practical examples using STM32/ESP32-style embedded thinking

## Background

I am a B.Sc. Automatic Control and Robotics graduate working as a Junior Automation & Embedded Systems Engineer. My technical background includes STM32, ESP32, embedded systems, industrial automation, PCB debugging/testing, communication protocols, ROS2, and robotics.

## Repository Philosophy

- No fake industrial claims
- No exaggerated expertise
- Concepts explained step by step
- Practical embedded examples included
- Notes evolve progressively
- Recruiter-friendly but technically honest
