# STM32 Safety Mechanisms

STM32 systems can include several safety-oriented mechanisms.

## Examples

- independent watchdog
- window watchdog
- brownout reset
- clock security system
- CRC peripheral
- memory protection unit on supported MCUs
- ADC watchdog
- communication error checks

## Important Note

Using a mechanism does not automatically make a system ISO 26262 compliant. The mechanism must be linked to safety requirements and verified.
