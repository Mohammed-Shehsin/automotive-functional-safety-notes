# Safety and Cybersecurity

Functional safety and cybersecurity are increasingly connected.

## Main Idea

Functional safety focuses on malfunctioning behavior. Cybersecurity focuses on malicious or unauthorized actions.

## Example

If an attacker manipulates braking communication, the result may become a functional safety hazard.

## Embedded Perspective

Safety-related embedded systems should consider secure communication, authentication, software update integrity, diagnostic access control, and separation of critical functions.

## Personal Observation

A safe system can become unsafe if security weaknesses allow external manipulation.
