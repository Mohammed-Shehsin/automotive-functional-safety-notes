# Learning Roadmap

## Stage 1 — Foundation
- Understand functional safety
- Learn ISO 26262 structure
- Understand ASIL, HARA, and safety lifecycle

## Stage 2 — Analysis Methods
- Study FMEA
- Study FTA
- Study dependent failure analysis
- Understand systematic vs random hardware failures

## Stage 3 — Embedded Safety
- Watchdog timers
- Diagnostics
- Fail-safe outputs
- Redundancy
- Sensor plausibility checks
- Startup self-tests

## Stage 4 — Automotive Embedded
- CAN/CAN FD basics
- ECU safety concepts
- Safety-oriented communication
- Error handling and degraded operation

## Stage 5 — Portfolio Projects
- Watchdog demo
- Sensor fault detection
- Safe-state LED controller
- ECU startup self-test
