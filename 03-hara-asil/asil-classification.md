# ASIL Classification

ASIL means Automotive Safety Integrity Level.

## Levels

- QM
- ASIL A
- ASIL B
- ASIL C
- ASIL D

ASIL D is the highest safety integrity level.

## Parameters

ASIL is determined using severity, exposure, and controllability.

## Embedded Perspective

Higher ASIL levels usually require stronger diagnostics, verification, redundancy, fault handling, and documentation.
