# FMEA — Failure Mode and Effects Analysis

FMEA is a structured method used to identify possible failure modes and their effects.

## Basic Questions

- What can fail?
- What causes the failure?
- What happens if it fails?
- How can it be detected?
- How can the risk be reduced?

## Embedded Example

Failure mode: ADC sensor reading stuck at maximum value.

Possible effect: Incorrect control output.

Possible mitigation: Range check, plausibility check, timeout detection.
