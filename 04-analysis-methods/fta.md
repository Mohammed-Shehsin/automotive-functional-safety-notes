# FTA — Fault Tree Analysis

FTA is a top-down analysis method. It starts with an unwanted event and analyzes possible causes.

## Example Top Event

Motor turns on unexpectedly.

## Possible Causes

- software logic error
- stuck output driver
- corrupted command
- communication fault
- failed safety interlock

## Engineering Value

FTA helps understand combinations of faults that may lead to a hazardous event.
